How to submit a quote of the day!

  1. Upload the image that you want to be posted on Instagram
  2. Edit the photo, you can add text to the photo if you would like!
  3. Hit save once done editing!!!! **It will not work if you do not do that**
  4. Enter in what you want the caption to be
  5. Hit submit!
